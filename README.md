# Responsive Personal Portfolio --- README

## Overview

This repository contains a modern, fully responsive **Personal Portfolio
Website** developed using **HTML5** and **advanced CSS3**. The design
follows a neon--dark theme with elegant gradients, smooth animations,
and accessibility‑focused components. This project was created as part
of:

## Features

### ✔ Modern & Responsive UI

-   Built with CSS Grid and Flexbox\
-   Works seamlessly on mobile, tablet, and desktop

### ✔ Neon Dark Aesthetic

-   Gradient backgrounds\
-   Glow effects\
-   Soft shadows and glassmorphism cards

### ✔ Interactive Elements

-   Hover effects\
-   Smooth transitions\
-   Visual depth through layered UI elements

### ✔ Accessibility Support

-   Skip‑to‑content link\
-   Semantic structure\
-   Focus states for keyboard navigation

### ✔ Well‑Structured Sections

-   **Hero Section** --- Intro + call-to-action\
-   **About** --- Education, focus area, interests\
-   **Skills** --- Key frontend skills\
-   **Projects** --- Three featured mini‑projects\
-   **Contact** --- Form, social links, and downloadable CV\
-   **Footer** --- Navigation shortcuts

## How to Use

1.  Download all project files.
2.  Ensure `Java.html` and `try.css` remain in the same directory.
3.  Open **Java.html** in any browser (Chrome, Firefox, Edge, etc.).
4.  All styling will load automatically.
